## 📋 What is VitalGuard?

VitalGuard is a user-friendly application designed to help hospitals and ICU staff monitor patient vital signs remotely. It tracks health data in real time and uses a smart system to predict risks before they become critical. This lets healthcare workers act quickly and keep patients safer. The full system shows live data on an easy-to-use dashboard.

You don’t need to be a tech expert to use this. Just download, install, and start monitoring quickly on any Windows PC.

## 🧰 Technical Details (Optional for Users)

The system uses a machine learning model called XGBoost for risk prediction. It processes patient vitals to estimate health changes quickly. The user interface runs on a React dashboard that updates instantly as new data arrives.

Data moves through four layers:

1. ICU Command Center (main dashboard)  
2. Smart Triage Core (AI model)  
3. Patient Intelligence Panel (individual data)  
4. Intelligent Alert Engine (notifications)

All processing happens behind the scenes. You just interact with a clear and simple app.
